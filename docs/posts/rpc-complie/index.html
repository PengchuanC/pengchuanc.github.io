记录GRPC在各种语言下的编译方式

Python

Python中的编译方式

假设当前工作目录为protobuf，待编译文件为hello.proto

> cd protobuf
> python -m grpc_tools.protoc --python_out=. --grpc_python_out=. -I. hello.proto

Go

Golang中的编译方式

> cd protobuf
> protoc --go_out=plugins=grpc:. hello.proto
